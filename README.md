<h1>CPU Scheduling Simulator</h1>
<p>A web-based CPU Scheduling Visualizer that allows users to simulate major CPU scheduling algorithms and view Gantt Charts. Simple UI and fast performance.</p>

<hr>

<h2>üöÄ Features</h2>
<ul>
  <li>Supports 5 algorithms:
    <ul>
      <li>FCFS</li>
      <li>SJF</li>
      <li>SRTF</li>
      <li>Priority Scheduling</li>
      <li>Round Robin (RR)</li>
    </ul>
  </li>
  <li>Real-time Gantt Chart generation</li>
  <li>Clean minimal dark UI</li>
  <li>Flask REST API backend</li>
  <li>Matplotlib-based chart rendering</li>
  <li>Frontend using HTML, CSS, JS</li>
  <li>Deployable on Vercel (Serverless Python)</li>
</ul>

<h2>‚ñ∂Ô∏è Running the Project Locally</h2>

<h3>1. Clone the repository</h3>
<pre>git clone https://github.com/your-repo/cpu-scheduler
cd cpu-scheduler</pre>

<h3>2. Install backend dependencies</h3>
<pre>cd api
pip install -r requirements.txt</pre>

<h3>3. Run Flask server</h3>
<pre>python app.py</pre>
<p>Backend starts at: <code>http://localhost:5000</code></p>

<h3>4. Open frontend</h3>
<p>Open the file: <code>frontend/index.html</code></p>
